Сапёр.

Управление:

 ПКМ - поставить флаг
 ЛКМ - открыть клетку/клтки (при клике по цифре вокруг которой столько же бомб какой её номер)
 R - рестарт


Настройки:
	
 Откройте файл config.config
 настройте размер окна по осям x и y
 введите количество мин

 Если вдруг файл отсутствует или был повреждён запустите minesweeper.exe и он починит.
